# Laravel sailによる、Docker環境の構築(dockerは既にインストールされているものとする)
構築パターンは以下の2つ。
1. １から構築する
2. git cloneしたものを動かす
   
## パターン１
1. cd {任意のディレクトリ}
2. curl -s https://laravel.build/{プロジェクト名} | bash
（飯食ってきていいくらい時間かかった）
3. ./vendor/bin/sail up -d
```
ex)  
$ curl -s https://laravel.build/intro_entiy | bash
$ ./vendor/bin/sail up -d
```
